## How to Run This Project

1. Clone the repository:
   ```bash
   git clone https://github.com/VsalCode/fgo24-go-introduction.git
   ```

2. Navigate to the project folder:
   ```bash
   cd fgo24-go-introduction
   ```

3. Run the program
   ```bash
   go run main.go
   ```
